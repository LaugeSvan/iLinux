# Contributing to iLinux

Thanks for your interest in contributing to iLinux.

This project is licensed under the GNU General Public License v3.0 or later.
By contributing, you agree that your contributions will be licensed under
the same terms.

## What we accept

Good contributions include:
- New GTK themes
- New icon packs
- New cursor themes
- Wallpapers
- GNOME compatibility improvements
- Script safety improvements
- Code cleanup and refactors

All features must remain optional and interactive.

## What we do not accept

- Closed-source dependencies
- Proprietary assets
- Apple proprietary software
- Hard-coded distro-specific hacks
- Anything that breaks non-recommended installs

## Coding guidelines

- Bash scripts must be readable
- Use functions, not copy-paste blocks
- Avoid destructive commands
- Always respect user choice
- Assume the user cares about their system

## Licensing

All contributions must be GPLv3 compatible.
If you cannot or do not want to license your work under GPLv3, do not contribute.
When submitting assets (themes, icons, wallpapers), ensure you have the right to
distribute them under GPLv3.
